===========================================================
                    TEAMFLOW - PROJECT OVERVIEW
===========================================================

Hi! Welcome to TeamFlow. 

This is a professional task management app built to help 
teams stay organized. It works like a digital office where 
you can track projects, tasks, and team progress in real-time.

-----------------------------------------------------------
1. CORE FEATURES
-----------------------------------------------------------

* Project Organization: 
  Create different projects for different goals (e.g., Website 
  Redesign, Marketing Campaign).

* Kanban Board: 
  Manage tasks using a "To Do", "In Progress", and "Done" 
  workflow. You can drag and drop tasks as you work.

* Task Details: 
  Set deadlines, pick a priority (Low, Medium, High), and 
  assign tasks to specific team members.

* Real-Time Activity Log: 
  The app keeps a history of everything that happens. You can 
  see who created a task or moved it to "Done" and when.

* Secure Login: 
  We use professional OTP (One-Time Password) verification 
  to keep your data safe and ensure only team members can enter.

-----------------------------------------------------------
2. TECHNOLOGY STACK
-----------------------------------------------------------

* Frontend: React (with Tailwind CSS for a modern design)
* Backend: Node.js & Express
* Database: PostgreSQL (using Prisma ORM)
* Security: JWT (JSON Web Tokens) and Cookie-based Auth
